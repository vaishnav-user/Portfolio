# Responsive Portfolio Website

## Project Description
This is a responsive personal portfolio website developed using HTML and CSS as part of a mini project.  
The website includes a hero section, skills, projects, and a contact form, and adapts smoothly to mobile, tablet, and desktop screen sizes.

The project focuses on applying core web development concepts and following a proper GitHub workflow.

---

## Tech Stack
- HTML5  
- CSS3  
- Flexbox  
- CSS Grid  
- Media Queries  
- CSS Variables (Dark / Light Mode)  
- Git & GitHub  
- GitHub Pages
